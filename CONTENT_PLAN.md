# Контент-план DarkCase

## Цель
Регулярное пополнение базы кейсов для поддержания интереса пользователей и роста аудитории.

## Стратегия пополнения

### Еженедельное пополнение
- **Количество**: 2-3 новых кейса в неделю
- **Приоритет**: Актуальные дела, недавно раскрытые, громкие судебные процессы
- **Источники**: 
  - Новостные агентства (BBC, CNN, Reuters)
  - Криминальные подкасты и документальные фильмы
  - Официальные источники правоохранительных органов
  - Криминальные базы данных (FBI, Interpol)

### Ежемесячное пополнение
- **Количество**: 10-15 новых кейсов в месяц
- **Фокус**: 
  - Разнообразие стран (не только США)
  - Разные типы преступлений
  - Исторические дела
  - Холодные дела, которые были раскрыты

### Квартальное обновление
- **Обновление существующих кейсов**: Добавление новой информации о раскрытых делах
- **Специальные коллекции**: Тематические подборки (например, "Дела 2020-х", "Раскрытые через ДНК")
- **Улучшение контента**: Добавление новых фактов, обновление хронологии

## Категории контента

### По типу преступления
1. **Серийные убийцы** (Serial Killers)
2. **Таинственные исчезновения** (Disappearances)
3. **Убийства знаменитостей** (Celebrity Murders)
4. **Семейные убийства** (Family Murders)
5. **Полицейские убийства** (Police Killings)
6. **Торговля людьми** (Human Trafficking)
7. **Терроризм** (Terrorism)
8. **Киберпреступления** (Cybercrime)
9. **Корпоративные преступления** (Corporate Crime)
10. **Политические убийства** (Political Assassinations)

### По стране
- США (приоритет - большая база)
- Великобритания
- Россия
- Германия
- Франция
- Италия
- Австралия
- Канада
- Мексика
- Бразилия
- Япония
- Южная Корея

### По статусу
- **Раскрыто** (Solved) - 40%
- **Нераскрыто** (Unsolved) - 35%
- **Холодное дело** (Cold Case) - 25%

## Критерии отбора кейсов

### Обязательные требования
1. **Достоверность**: Только подтвержденные факты из официальных источников
2. **Этичность**: Уважение к жертвам и их семьям
3. **Полнота информации**: Достаточно данных для создания полного описания
4. **Юридическая чистота**: Избегание клеветы и неподтвержденных обвинений

### Приоритетные критерии
1. **Публичный интерес**: Дела, получившие широкое освещение
2. **Уникальность**: Необычные обстоятельства или методы
3. **Образовательная ценность**: Дела, демонстрирующие работу правоохранительных органов
4. **Актуальность**: Недавние дела или новые раскрытия старых

## Процесс добавления контента

### Шаг 1: Исследование
- Сбор информации из надежных источников
- Проверка фактов
- Сбор изображений (с соблюдением авторских прав)

### Шаг 2: Создание контента
- Написание описания на русском языке
- Создание хронологии событий
- Выделение ключевых фактов
- Определение категорий и тегов

### Шаг 3: Редактирование
- Проверка на ошибки
- Проверка этичности контента
- Оптимизация для SEO

### Шаг 4: Публикация
- Добавление в базу данных
- Обновление категорий
- Публикация в приложении

## Метрики успеха

### Количественные
- Количество новых кейсов в месяц
- Охват различных категорий
- Географическое разнообразие

### Качественные
- Время, проведенное пользователями на страницах кейсов
- Количество добавлений в избранное
- Отзывы пользователей

## Календарь контента (пример)

### Январь 2024
- Неделя 1: 2 новых кейса (США, Великобритания)
- Неделя 2: 3 новых кейса (разные типы преступлений)
- Неделя 3: 2 новых кейса + обновление 1 старого
- Неделя 4: 3 новых кейса (специальная коллекция "Дела 2023")

### Февраль 2024
- Неделя 1: 2 новых кейса
- Неделя 2: 3 новых кейса
- Неделя 3: 2 новых кейса + обновление 2 старых
- Неделя 4: 3 новых кейса (специальная коллекция "День влюбленных - дела о страсти")

## Источники контента

### Основные источники
1. **Официальные источники**
   - ФБР (FBI)
   - Интерпол (Interpol)
   - Национальные полицейские агентства
   - Судебные протоколы

2. **Новостные агентства**
   - BBC News
   - CNN
   - Reuters
   - Associated Press
   - РИА Новости (для российских дел)

3. **Специализированные ресурсы**
   - Криминальные подкасты
   - Документальные фильмы
   - Криминальные базы данных
   - Книги о криминале

### Проверка информации
- Все факты проверяются минимум из 2 независимых источников
- Приоритет официальным источникам
- Избегание спекуляций и теорий заговора без подтверждения

## Этические принципы

1. **Уважение к жертвам**: Никогда не сенсационализировать страдания
2. **Точность**: Только проверенные факты
3. **Конфиденциальность**: Защита личной информации несовершеннолетних
4. **Баланс**: Представление всех сторон дела
5. **Образовательная цель**: Фокус на понимании, а не на развлечении

## Автоматизация

### Планируемые инструменты
- Система уведомлений о новых делах
- Автоматический сбор информации из новостных источников
- Шаблоны для быстрого создания контента
- Система проверки дубликатов

## Ответственность

- **Контент-менеджер**: Отвечает за создание и публикацию контента
- **Редактор**: Проверяет качество и этичность контента
- **Юрист**: Проверяет юридическую чистоту контента
- **Исследователь**: Собирает и проверяет информацию
