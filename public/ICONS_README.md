# Иконки для PWA

Для работы Service Worker и PWA нужны иконки приложения.

## Требуемые размеры:
- `icon-192.png` - 192x192 пикселей
- `icon-512.png` - 512x512 пикселей

## Как создать иконки:

### Вариант 1: Онлайн генератор
1. Используйте https://realfavicongenerator.net/ или https://www.pwabuilder.com/imageGenerator
2. Загрузите ваше логотип/иконку
3. Скачайте сгенерированные иконки
4. Поместите их в папку `public/`

### Вариант 2: Вручную
1. Создайте изображение 512x512 пикселей с логотипом DarkCase
2. Сохраните как `icon-512.png`
3. Уменьшите до 192x192 и сохраните как `icon-192.png`
4. Поместите оба файла в папку `public/`

## Временное решение:
Пока иконки не созданы, Service Worker будет работать, но PWA манифест будет неполным.
Можно использовать placeholder изображения или временно отключить требование иконок в `vite.config.ts`.
